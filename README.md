# SHE CAN FOUNDATION 🌸

A simple and responsive NGO website created using HTML and CSS.
This project is designed to spread awareness about women empowerment and showcase the mission and vision of **SHE CAN FOUNDATION**.

---

## 📌 Project Overview

The website contains:

* Homepage
* About Section
* Foundation Introduction
* Women Empowerment Content
* Call-to-Action Buttons
* Attractive Background Image
* Footer Section

This project is beginner-friendly and suitable for practicing:

* HTML
* CSS
* Web Design Basics
* GitHub Hosting

---

## 🛠️ Technologies Used

* HTML5
* CSS3
